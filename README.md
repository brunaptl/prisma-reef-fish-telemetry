# PRISMA Systematic Review – Reef Fish Acoustic Telemetry

This repository contains the search and screening records used in the systematic review **“Habitat use by reef fishes based on acoustic telemetry: a global analysis.”**

The materials currently available include:

- **Web of Science search logs** (search strings, keywords, dates, and results)
- **Rayyan screening reports** (criterias, filters, screening decisions and exclusion reasons)

These files support the transparency and reproducibility of the review following the **PRISMA 2020 guidelines**.
